# Physics Notes (JupyterLite)

This site contains interactive, browser-based physics notes powered by **JupyterLite** and **Pyodide**.

### Features
- Notes written in Markdown + LaTeX  
- Real runnable Python code directly in the browser  
- Physics simulations (interference, diffraction, mechanics, EM)  
- No installation required  
- Fully static website hosted on GitHub Pages
